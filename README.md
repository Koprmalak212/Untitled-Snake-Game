# 🐍 Untitled Snake Game

A simple but polished version of the classic Snake game built for a school project.  
Focus is on smooth gameplay, clean visuals, and a fun arcade experience.

---

## 🎮 Features

- Classic snake gameplay
- Smooth movement and controls
- Score tracking system
- Increasing difficulty over time
- Clean and minimal UI

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| W / ↑ | Move Up |
| S / ↓ | Move Down |
| A / ← | Move Left |
| D / → | Move Right |
| ESC | Pause Game |

---

## 🚀 Installation

1. Clone the repository  
2. Open the project in your IDE  
3. Build and run the game  
4. Enjoy playing 🐍
